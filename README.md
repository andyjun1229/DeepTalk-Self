# 深度自我认知系统

**让AI像顶级访谈主持人一样，帮你拆解自己**

> 不是心理测试。是仿生名人访谈的深度自我对话。

---

## 这是什么

一套基于**名人访谈方法论复刻**的自我认知框架。

核心思路不是设计"更好的问题"，而是复刻那些顶级访谈主持人（杨澜、易立竞、许知远等）的追问逻辑——让AI扮演一个不评判、不打断、不怕得罪人的访谈者，通过结构化追问帮你拆解自己的决策模式、人际盲点、价值观和深层动机。

## 核心方法：名人访谈仿生

本系统不是从心理学理论推导出的框架，而是从优秀名人访谈节目中提炼出追问逻辑后，逆向工程出的可执行对话系统。

**我们模仿了三个特征：**

1. **不懂就问** — 杨澜式的坦诚。主持人不装懂，AI天然适合。
2. **矛盾摆在台面上** — 易立竞式的追问。把你前后不一致的地方摆出来，让你自己看。
3. **不给答案，让你自己找到答案** — 许知远式的方向。"你是怎么变成这样的"，而不是"你应该怎么办"。

**追问链设计（每个核心问题带3-4层追问路径）：**

```
核心问题 → 影响溯源（谁/什么影响了你）
         → 后悔检验（再选一次会怎样）
         → 模式识别（这个模式在其他阶段出现过吗）
```

每条追问路径都有明确的目的——不是随意追问，是定向拆解某个认知模块。

## 框架结构

| 维度 | 目标 | 追问数 |
|------|------|--------|
| 决策逻辑 | 拆解你的决策权重、风险偏好、影响源 | 9 |
| 人际交往 | 拆解你的沟通风格、信任标准、冲突模式 | 9 |
| 学习成长 | 拆解你的学习方式、能力迁移模式 | 9 |
| 价值观 | 拆解你的价值排序、深层驱动力 | 9 |
| 自我认知 | 识别你的自我评价偏差、盲点 | 12 |
| **总计** | | **39个核心问题 + 约80条追问路径** |

## 谁适合用

- 职业转型期的人（帮你分清"真的不适合"还是"习惯性想逃"）
- 想了解自己的人（不是贴标签，是拆解）
- 找不到方向的人（不是给答案，是让你自己看到方向）
- 准备做重大决定的人（追问链逼你从多角度看同一件事）
- 觉得自己该改变但动不了的人（先找到"为什么改不了"）

## 怎么用

**方式一：AI对话（推荐）**

把 `SKILL.md` 中的提示词发给AI助手，说"开始深度访谈"。AI会按框架逐层提问，你回答，它会基于你的回答调整追问方向。每天7-8个问题，一周完成。

**方式二：离线自问自答**

下载 `interview-framework.md`，按问题列表自问自答并记录答案，之后可以找人讨论或让AI分析。
