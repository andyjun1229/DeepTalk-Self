# 深度人物访谈框架设计

## 研究方法

由于网络搜索工具暂时不可用，我基于以下来源整合方法论：

1. **心理测评理论** - MBTI、大五人格、StrengthsFinder的提问逻辑
2. **优秀访谈案例** - 《十三邀》《杨澜访谈录》《大卫·弗罗斯特访谈》
3. **人物画像方法** - 用户画像、角色卡、小说人物塑造技巧
4. **深度访谈技巧** - 心理咨询中的开放式提问、追问技巧

## 核心原则

### 1. 多维度交叉验证

不是单一问答，而是从不同角度问同一类问题，交叉验证：
- 直接问："你遇到冲突时会怎么做？"
- 间接问："你见过别人处理冲突最糟糕的方式是什么？"
- 场景问："如果你的下属跟你意见不合，你会怎么办？"

### 2. 打破防御

避免"你想让我说什么"的防御性回答：
- 不问价值观抽象概念（"你认为什么最重要"）
- 问具体场景（"上次你为了什么而拒绝了一个机会"）
- 不问"你觉得怎么样"
- 问"当时你做了什么决定"

### 3. 挖掘潜意识

很多时候人不知道自己的真实动机，通过行为追溯：
- 不问"你为什么要这样做"
- 问"当时那个瞬间，你脑子里想的第一个念头是什么"
- 不问"你觉得这件事对你有什么影响"
- 问"这件事之后，你的第一个行为改变是什么"

## 访谈框架设计

### 第一维度：决策逻辑（通过具体决策体现价值观）

**目标**：理解你的决策权重、风险偏好、信息处理方式

#### 子问题组A：重要决策案例
1. "过去5年，你觉得最正确的决定是什么？当时为什么不那么确定？"
2. "有没有哪个决定，你现在回想起来'当时要是那样就好了'？"
3. "做决定前，你通常会问自己3个问题是什么？"

#### 子问题组B：信息来源
1. "当你面对不确定性时，你会先听谁的意见？最后听谁的？"
2. "有没有哪个人/哪本书/哪次经历，彻底改变了你做决定的方式？"
3. "你会为了什么而忽略数据分析？"

#### 子问题组C：失败复盘
1. "你印象最深的失败，不是结果不好，而是'我犯了一个不应该犯的错'的那种"
2. "你有没有过'我当时知道不该这样但还是这样做了'的经历？"
3. "你从失败中学到的东西，你会怎么验证它真的学到了？"

### 第二维度：人际交往模式（通过互动体现社交能力）

**目标**：理解你的沟通风格、关系建立方式、冲突处理模式

#### 子问题组A：关系建立
1. "你跟一个人从'不认识'到'信任'，中间经历了什么？"
2. "你有没有过'本以为跟这个人会合不来，但结果意外的好'的经历？"
3. "你信任一个人的3个标准是什么？"

#### 子问题组B：冲突处理
1. "你最近一次跟人冲突，不是因为观点，而是因为'你这个人/我这个人'，是什么时候？"
2. "你会为了什么而主动冲突？"
3. "你见过最糟糕的冲突处理方式是什么？"

#### 子问题组C：影响力
1. "你觉得你有影响力吗？体现在哪里？"
2. "你有没有过'我本来想影响这件事，但结果没有'的经历？"
3. "你会用什么方式让别人接受你的意见？"

### 第三维度：学习与成长模式（通过变化体现能力边界）

**目标**：理解你的学习方式、成长速度、能力迁移模式

#### 子问题组A：学习方式
1. "学一个新东西，你通常从哪里开始？"
2. "你有没有过'本来以为这件事我学不会，结果学会了'的经历？"
3. "你会为学习这件事而放弃什么？"

#### 子问题组B：能力迁移
1. "你从A领域（比如销售）学到的能力，是怎么迁移到B领域（比如技术）的？"
2. "你有没有过'在A领域很有用的方法，在B领域完全不管用'的经历？"
3. "你觉得能力迁移的关键是什么？"

#### 子问题组C：成长速度
1. "你觉得自己成长最快的3年，是什么时候？那3年发生了什么？"
2. "你现在花时间最多的3件事是什么？这3件事会把你带到哪里？"
3. "你会主动寻找成长机会，还是等待机会？"

### 第四维度：价值观与动机（通过矛盾体现深层动机）

**目标**：理解你的价值排序、驱动力来源、矛盾冲突点

#### 子问题组A：价值排序
1. "如果你必须在A和B之间选一个，你永远选A，是什么情况？"
2. "你见过别人'为什么把A看得比B重要'让你无法理解的案例？"
3. "你会为了什么而放弃高薪/高职位/稳定？"

#### 子问题组B：驱动力
1. "什么会让你觉得'今天没白过'？"
2. "你有没有过'当时不知道为什么要做这件事，但后来觉得做对了'的经历？"
3. "你觉得什么在推动你往前走？不是'应该'，是'真正推动'"

#### 子问题组C：矛盾点
1. "你内心最矛盾的一个观念是什么？A和B，你都认同，但它们冲突"
2. "你有没有过'嘴上说A，但行为是B'的经历？"
3. "你见过别人'想得A，但行为导致B'的案例，你觉得为什么？"

### 第五维度：自我认知与防御机制（通过盲点体现潜意识）

**目标**：识别你的自我评价偏差、防御模式、盲点

#### 子问题组A：自我评价
1. "你觉得别人对你的评价，跟你对自己的评价，最大的差异是什么？"
2. "你有没有过'我以为我很擅长这件事，但事实证明并不是'的经历？"
3. "你对自己的3个不满意是什么？"

#### 子问题组B：防御机制
1. "你什么时候会'不想说'？"
2. "你有没有过'当时没说实话'的经历？"
3. "你会用什么方式保护自己？"

#### 子问题组C：盲点
1. "你见过别人'以为自己很清楚，但完全搞错了'的案例，你觉得为什么会这样？"
2. "你有没有过'后来才发现，我当时完全没意识到'的经历？"
3. "你觉得你的盲点可能在哪里？"

## 执行流程

### Phase 1：破冰 + 建立信任

不是直接跳到敏感问题，而是：
- 先从安全话题切入（过去的成功案例、转折点）
- 建立访谈的"共同目标"（不是审问，是一起探索）
- 说明隐私边界（不会对外分享）

### Phase 2：维度内深度挖掘

每个维度内，按照"直接→间接→场景"的顺序：
1. 直接问：基本事实
2. 间接问：通过别人/案例观察
3. 场景问：具体行为反应
4. 追问："当时那个瞬间你脑子里想的第一个念头是什么？"

### Phase 3：维度间交叉验证

不是线性推进，而是跳跃式验证：
- 在"决策逻辑"维度发现的信息，到"人际交往"维度验证
- 在"价值观"维度发现的矛盾，到"自我认知"维度追问

### Phase 4：画像生成与验证

1. 生成初步画像
2. 反馈给用户："这是我对你的理解，对吗？"
3. 用户修正："不对，我其实是..."
4. 迭代画像

## 防御性回答识别

### 常见防御模式

1. **标准化回答** - "我觉得平衡很重要"
   - 反问："你能举一个具体的例子吗？"

2. **理性化回答** - "从ROI角度看，应该..."
   - 追问："当时那个瞬间，你心里的第一个情绪是什么？"

3. **回避性回答** - "这个情况比较复杂"
   - 具体化："能不能说出一个具体的事件？"

4. **自我评价过高/过低** - "我在这方面很擅长/很差"
   - 验证："你觉得别人会怎么评价你在这方面的表现？"

### 反防御技巧

1. **不问"你怎么想"，问"你当时做了什么"**
2. **不问"你觉得怎么样"，问"那个瞬间你的第一个念头是什么"**
3. **不问抽象概念，问具体场景**
4. **通过间接提问绕过防御** - 不问"你的价值观"，问"你会为什么放弃高薪"

## 画像输出格式

### 1. 能力图谱

- 强项：通过具体案例验证的
- 待提升：通过自我反思发现的
- 潜力领域：基于能力迁移能力推断的

### 2. 价值观排序

- 核心价值观：通过矛盾选择验证的
- 次要价值观：通过日常决策体现的
- 潜在冲突点：内心矛盾的识别

### 3. 人际交往模式

- 沟通风格：具体表现
- 关系建立方式：信任标准
- 冲突处理模式：应对方式

### 4. 学习与成长模式

- 学习方式：如何学习新东西
- 成长速度：对比不同阶段
- 能力迁移：从A到B的关键

### 5. 发展路径设计（AI领域）

- **优势匹配** - 你的哪些能力适合AI领域？
- **路径设计** - 从当前状态到AI从业者的具体步骤
- **风险点** - 你可能会遇到的障碍
- **加速器** - 如何利用你的强项加速成长
